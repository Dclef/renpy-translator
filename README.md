# Renpy tl文件翻译工具

## 代码思路

将tl源文件移动到代码根目录，将所有文本合并到一个文本中并进行标记并将文本放在游览器，通过游览器的翻译功能进行翻译，最后进行合并到源文件中。思路很简单，有意者可进行改进。

## How to use

请先运行代码

### 1.移动rpy文件到当前文件夹

按1

### 2.制作翻译文件

按3

### 3.将google trans文件放在游览器进行翻译

如图![](https://cdn.jsdelivr.net/gh/dclef/CDN/renpy/renpy-trans.png)

### 4.分割

按4

### 5.合并

按5

### 6.合并源文件

按2

请自行删除googletrans等文件
